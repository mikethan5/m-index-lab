# m-index-lab

## 项目简介

`m-index-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的是各种实验性、临时性或功能性的单页 HTML 文件，不针对特定域名或网站，旨在提供一个集中存放和展示这些页面源码的场所。

## 目录说明

仓库根目录下直接放置各个独立的 HTML 文件，每个文件代表一个完整的、可直接运行的页面。部分文件可能附带同名的资源文件夹（如 `css`、`js`、`img` 等），用于存放该页面所需的静态资源。

目录结构示例：

```
/
├── page-a.html
├── page-b.html
├── page-c/
│   ├── index.html
│   └── assets/
└── README.md
```

## 页面归档说明

- 每个 HTML 文件均为独立页面，可直接在浏览器中打开。
- 页面内容可能包含简单的交互、数据展示或工具功能，但不会涉及任何商业或营销用途。
- 所有页面均基于纯前端技术实现，不依赖后端服务。
- 部分页面可能引用外部 CDN 资源（如字体、图标库等），请确保网络连接正常。

## 维护说明

- 本仓库主要用于个人归档和学习记录，不定期更新。
- 欢迎提交 Issue 或 Pull Request 进行改进或修复。
- 任何提交的页面应保持独立、无恶意代码，并遵守开源社区规范。
- 本仓库不提供任何形式的在线服务承诺，页面可能因时间推移而失效或不再维护。
